# 💱 Currency Converter

A simple and responsive Currency Converter web application built using **HTML, CSS, and JavaScript**. It fetches real-time exchange rates from an API and allows users to convert between different currencies instantly.

---

## 🚀 Features

- 🌍 Convert between multiple currencies
- 🔄 Swap currencies with one click
- 📊 Real-time exchange rates using API
- 🇺🇸 Dynamic country flags based on selection
- ⚡ Instant calculation with clean UI
- ❤️ Interactive UI elements (like button animation)

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Exchange Rate API
